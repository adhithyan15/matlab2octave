Welcome to Matlab2Octave.

Hello Everyone. Thanks for taking time to visit my simple project Matlab2Octave. I program a lot in a different languages. One of it is Matlab. Octave is a free software that is almost compatible with matlab. You need some syntax modifications to make the Matlab code work in Octave. For people who cannot afford a Matlab license, Octave is their best bet. This compiler converts Matlab code into syntactically correct Octave code.
