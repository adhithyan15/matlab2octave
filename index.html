Welcome to Matlab To Octave Converter.

Hello Everyone. Thanks for taking time to visit my simple project matlab2octave. I program a lot in a different languages. One of it is matlab. Octave is a free software that is almost compatible with matlab. You need some syntax modifications to make the matlab code work in octave. For people who cannot afford a matlab license, Octave is their best bet. Instead of manually changing the syntax of your matlab file to make it octave compatible, this project achieves it automatically.

This is a very simple project with not more than 1000 lines of code combined. This is also my first open source project that I actually used.
